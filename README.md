<h3><웹 프로젝트 : 영화 소개 및 평점 부여, 게시판 작성></h3>

- 기본적인 기능
  - 로그인, 로그아웃, 회원가입, 회원정보조회, 아이디찾기, 비밀번호 찾기, 회원탈퇴
- Home
  - 로그인 전
    - 조회수 1~3등까지 표시
  - 로그인 후
    - 평점 부과에 따른 관심 영화를 HOME에서 바로 보기 가능
    - 조회수 1~3등까지 표시
- 게시판
  - summernote 사용
  - 수정, 삭제
  - 추천 (한 id 당 1번만 가능)
  - 조회순, 최신순 정렬 선택 가능
  - 미로그인 : 게시판 작성, 추천 불가 
- 장르별 카테고리
